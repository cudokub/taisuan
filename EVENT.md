## ประเด็นไต่สวน (ข้อกล่าวหา)

**ภาพรวม:** การทำงานที่ไม่มีประสิทธิภาพและไม่โปร่งใสของ กกต.

### ประเด็นปัจจุบัน (เน้นเป็นพิเศษ)
1. **ผลเลือกตั้งไม่ครบ 100%** — ผ่านวันเลือกตั้งมา 2 สัปดาห์ ยังไม่รายงานตัวเลขผู้มาใช้สิทธิและผลการเลือกตั้งครบ
2. **ปัญหาระบบ Barcode** — ปัญหาการใช้ Barcode ในกระบวนการเลือกตั้ง
3. **ข้อขัดแย้งทางกฎหมายและรัฐธรรมนูญ** — ประเด็นถกเถียงสาธารณะเรื่องความชอบด้วยกฎหมาย

## Sequence — ลำดับการแสดง

| # | ก้อน | เวลา | รายละเอียด | คนดู |
|---|---|---|---|---|
| 1 | **เปิดเวที** | 5 นาที | MC เปิดงาน — mood เวทีมวย/ศาลไคฟง แนะนำทีมโจทก์-จำเลย บอกกติกา | |
| 2 | **จำเลยแถลง** | 5 นาที | หัวหน้า กกต. (เป๋า) แถลงสรุปผลการทำงาน — โทน: ทำงานดี เพอร์เฟกต์ สุจริต โปร่งใส ปิดด้วย "การแถลงจบแล้ว ขอเชิญสื่อและทุกท่านกลับบ้านได้" | |
| 3 | **โจทก์แย้ง** | 5 นาที | ทีมประชาชนแย้ง — เปิดด้วยพี่หนูหริ่ง | |
| 4 | **ถามตอบ ทีมโจทก์-จำเลย** | 40 นาที | โจทก์ทีละคนถาม (ประมาณ 3 คน) → จากนั้นเปิดถกแบบอิสระ | |
| 5 | **ถามตอบร่วมโดยประชาชน** | 30 นาที | คนดูยกมือถาม + กลุ่มเตรียมคำถามช่วยนำ | ✅ |
| 6 | **ช่วงท้ายชวนแสดงพลัง** | 3 นาที | MC + ผู้นำโจทก์ ถามคำถามชวนคนตอบแสดงพลัง → ประชาชนตะโกนคำตอบพร้อมกัน | ✅ |
| 7 | **ปิดเวที** | 2 นาที | | |
|  | **รวม** | **90 นาที** | | |

### การมีส่วนร่วมของคนดู
- **กลุ่มเตรียมคำถาม:** ประชาชนที่เตรียมมาล่วงหน้า ช่วยนำคำถามกับคนดูทั่วไป
- **คนดูทั่วไป:** ยกมือถามตอบได้
- **เสียง:** ใช้อุปกรณ์ติดตัว — ขวดน้ำ ปรบมือ เป่าปาก ส่งเสียงได้ตลอดงาน

### คำตะโกนปิดท้าย (ตัวอย่าง)
- "ติดคุก!"
- "หัวกรวย!"
- "เลือกตั้งแบบนี้ให้พ่อมรึงจิ้มมาเองเลยดีกว่า!"
